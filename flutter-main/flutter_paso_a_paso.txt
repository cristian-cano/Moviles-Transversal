|--------------------------------------------------------------------------------------------------------------------------------------------------------|
|-- ------  Instalar Visual Studio Code: ----------------------------------------------------------------------------------------------------------------|
|--------------------------------------------------------------------------------------------------------------------------------------------------------|

            01. Descargar 'Visual Studio Code': https://code.visualstudio.com/download.

            02. Instalar Visual Studio Code.

                    ⦿   En el asistente de instalación dejar marcado:   ☑ 'Agregar PATH…'.                    

            03. Abrir Visual Studio Code.
            
            04. Instalar las siguientes extensiones en Visual Studio Code:                    
                    
                    ⦿   Material Icon Theme
                    ⦿   Path IntelliSense
                    ⦿   Prettier - Code formatter

                    ⦿   Angular Essentials (Version 18)
                    ⦿   Angular Language Service
                    ⦿   Angular Snippets (Version 18)
                    ⦿   TypeScript Importer


|--------------------------------------------------------------------------------------------------------------------------------------------------------|
|-- ------  Instalar Node Js: ---------------------------------------------------------------------------------------------------------------------------|
|--------------------------------------------------------------------------------------------------------------------------------------------------------|

            01. Descargar 'node.js': https://nodejs.org/es/.

            02. Instalar node.js.


|--------------------------------------------------------------------------------------------------------------------------------------------------------|
|-- ------  PowerShell. Directivas: ---------------------------------------------------------------------------------------------------------------------|
|--------------------------------------------------------------------------------------------------------------------------------------------------------|

            01. Abrir el PowerShell de Windows
            
            01. Ejecutar el siguiente comando

                    PS C:\..\..> Get-ExecutionPolicy
                    Restricted

            NOTA:   ⦿   Si aparece 'Restricted', hay que cambiarla a 'Unrestricted' de la siguiente forma:

                    PS C:\..\..> Set-ExecutionPolicy -Scope CurrentUser

                    cmdlet Set-ExecutionPolicy en la posición 1 de la canalización de comandos
                    Proporcione valores para los parámetros siguientes:
                    ExecutionPolicy: Unrestricted

            02. Verificar la versión de 'node' y 'npm':

                    PS C:\..\..> node --version
                    PS C:\..\..> npm --version

|-- 01.01.  Creación del proyecto: ----------------------------------------------------------------------------------------------------------------------|
|--------------------------------------------------------------------------------------------------------------------------------------------------------|

            01. Crear la carpeta del proyecto en el computador (carpeta raíz):

                    C:\..\..\Desktop\angular

            02. Abrir con 'Visual Studio Code' la carpeta creada, es decir,'angular'.

            04. Abrir una terminal en Visual Studio Code ('Terminal / New Terminal' ó 'Ctrl + Shift + ñ')
            
            05. Instalar Angular escribiendo desde la terminal:
            
                    PS C:\..\..\Desktop\angular> npm install -g @angular/cli
             
            06. Verificar las versiones de node.js y angular desde la terminal:

                    PS C:\..\..\Desktop\angular> ng version

            07. Crear un proyecto de angular desde la terminal:

                    PS C:\..\..\Desktop\angular> ng new frontend

            NOTA:   ⦿   Seleccionar:

                    ✔ Do you want to create a 'zoneless' application without zone.js (Developer Preview)? y
                    ❯ CSS             [ https://developer.mozilla.org/docs/Web/CSS                     ]
                    ? Do you want to enable Server-Side Rendering (SSR) and Static Site Generation (SSG/Prerendering)? (y/N) y

            09. Cambiar el nombre de la terminal a 'frontend', seleccionándola en la parte inferior derecha y presionando F2 / Rename...
            
|--------------------------------------------------------------------------------------------------------------------------------------------------------|
|-- 01.02.  Ejecutar el servidor: -----------------------------------------------------------------------------------------------------------------------|
|--------------------------------------------------------------------------------------------------------------------------------------------------------|

            01. Ingresar a la carpeta 'frontend' desde la terminal:

                    PS C:\..\..\Desktop\angular> cd frontend

            02. Ejecutar el Servidor desde la terminal:

                    PS C:\..\..\Desktop\angular\frontend> ng serve -o

                    Would you like to share pseudonymous usage data about this project with the Angular Team

                    Watch mode enabled. Watching for file changes...

                    ➜  Local:   http://localhost:4200/

            NOTA:   ⦿   Con 'CTRL + C', se detiene la ejecución del servidor

            01. Crear una carpeta denominada 'backend' fuera del proyecto, ejemplo 'angular/backend'.
            
            02. Abrir una nueva terminal en vs code -> 'Terminal / New Terminal' ó 'Ctrl + Shift + ñ'.
                    
            03. Cambiar el nombre de la terminal a 'backend', seleccionándola en la parte inferior derecha y presionando F2 / Rename...                    
            
            04. Ingresar a la carpeta 'backend' desde la terminal:

                    PS C:\..\angular> cd backend

            05. Inicializar un proyecto de node.js desde la terminal:

                    PS C:\..\backend> npm init -y

            06. Instalar 'jsonserver' desde la terminal:

                    PS C:\..\backend> npm install json-server

            08. Descargar a la carpeta 'backend' la base de datos 'db.json' desde el siguiente enlace:             

                    https://github.com/ProfeAlbeiro/angular/blob/main/sources/db/db.json

            09. Ejecutar el servidor 'jsonserver' desde la terminal para asociarlo a la base de datos:

                    PS C:\..\backend> npx json-server db.json
